## âš™ï¸ Excel Functions Explained

The following core Excel functions are used throughout the workbook:

| Function | Category | Purpose | Example Use Case |
| :--- | :--- | :--- | :--- |
| **SUM** | Math/Trig | Calculates the total sum of a range of cells. | Implicitly used in Pivot Tables to total `Total Sales`. |
| **AVERAGE** | Statistical | Calculates the arithmetic mean of numbers. | Used in **Day 2 Task 2** to find the average student score. |
| **MAX** | Statistical | Returns the largest number in a set of values. | Used in **Day 2 Task 2** to find a student's `Highest score`. |
| **IF** | Logical | Checks whether a condition is met and returns one value if TRUE, and another value if FALSE. | Can be used in **retail\_sales\_dataset** to categorize commissions based on a sales threshold (e.g., `=IF(Sales>1000, "High", "Low")`). *This is an assumed logical use.* |
| **CONCATENATE / `&`** | Text | Joins several text strings or cell values into one text string. | Used in the **Transactions** sheet to combine `Total Sales` and `Product Category`. |
| **Multiplication (`*`)** | Math | Performs multiplication between two values. | Used in **retail\_sales\_dataset** to calculate `Total Sales` (`Quantity * Price`). |
